**BookMyStay – Full-Stack Hotel Booking Platform**
BookMyStay is a modern, full-stack hotel booking web application that allows users to discover hotels, view details, upload images, make secure online payments, and manage bookings — all through a smooth and intuitive user experience.
This project is designed to simulate a real-world travel booking system, similar to platforms like Booking.com or Airbnb, built using modern web technologies and best practices.

**Key Features**
**User Authentication**
Secure user registration and login
Authentication using JWT and HTTP-only cookies
Protected routes for authenticated users

**Hotel Management**
Add, edit, and view hotel listings
Manage hotel details such as name, location, price, and description
State management for seamless UI updates

**Image Uploads**
Upload and display hotel images
Proper handling of file uploads for real-world use cases

**Search, Sort & Filter**
Search hotels by name or location
Sort hotels by price or rating
Filter hotels to help users find the perfect stay quickly

**Online Payments**
Secure payment integration using Stripe
Real-time booking confirmation after successful payment

**Booking Management**
View booking history
Manage and track hotel bookings
Essential functionality for real booking platforms

**Recent Hotels on Home Page**
Display newly added hotels dynamically
Keeps the homepage fresh and engaging
